CSS3 buttons

This is a collection of buttons that show what is possible using CSS3 and other advanced techniques, while maintaining the simplest possible markup. These buttons look best in Chrome and Safari (especially on OSX). They look almost as good in Firefox, with all other browsers falling back to a less-styled button. If you use these buttons in the wild, drop me a note and let me know.

The latest code is located at: http://github.com/ubuwaits/css3-buttons

WebKit Experimental

Proof of concept buttons using WebKit-only features.

Bright on dark

Buttons styles suitable for using on dark backgrounds. These buttons also feature a radial gradient when clicked.

Classic buttons

This is the original set of buttons, showing a variety of CSS3 styles in different combinations. Study the code, then adjust to fit the context of where it's used.

Extended with Iconic

This set extends the functionality of the classic button set using icons embedded via @font-face. I am using P.J. Onori's free icon set, Iconic. You can read more about this technique on his blog.
